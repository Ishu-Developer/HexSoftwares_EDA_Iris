## Project Description
This project performs exploratory data analysis on the Iris dataset using Python.  
The goal is to explore the structure of the data, check for missing values, and visualize the distribution of each feature using plots and graphs.

## Dataset
The Iris dataset contains 150 samples and 5 columns:
- sepal_length
- sepal_width
- petal_length
- petal_width
- species

## Objectives
- Load and inspect the Iris dataset.
- Check for missing values and handle them if necessary.
- Analyze the dataset using descriptive statistics.
- Visualize the distribution of each feature.
- Create species-wise visualizations.
- Use a pairplot to understand relationships between features.

## Tools Used
- Python Language
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Findings
- The dataset has 150 rows and 5 columns.
- There are no missing values in the dataset.
- Each species has 50 samples, so the dataset is perfectly balanced.
- Setosa has the smallest petal measurements.
- Virginica has the largest petal measurements.
- Petal features separate the species better than sepal features.

## Output
The script generates:
- Feature distribution charts
- Species distribution pie charts
- Pairplot for feature relationships

## Conclusion
This project helped understand the Iris dataset structure and the differences between species through visual analysis.
